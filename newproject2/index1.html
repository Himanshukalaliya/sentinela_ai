API Documentation

Scan Endpoint

POST /api/scan

Request Body

{
    "content": "string (URL or text to scan)"
}

Response

{
    "status":  "HIGH RISK | SUSPICIOUS | SAFE",
    "score":   0-100,
    "reason":  "string",
    "domain":  "string",
    "safeBrowsing": "string",
    "safeBrowsingEnabled": true
}

Example

curl -X POST http://localhost:8081/api/scan \
  -H "Content-Type: application/json" \
  -d '{"content": "http://secure-login-example.com/verify"}'
